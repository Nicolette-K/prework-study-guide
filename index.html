HTML

  • There are two main elements in an html file, head and body
  • The head and body parent elements hold many child elements

  • The head element contains information about the webpage.
  • The head element is a container for meta elements which provide the browser the info needed to render the web page
  • The head element content is not visible to the user

  • The body element represents the visible content shown to the user.
  • The ul, ol, il elements are used for unordered lists, ordered lists, and list items respectively
  • Headers are represented using the h1-h6 tags, h1 being the largest text size and h6 being the smalllest
  • The id attribute is important in identifying elements
  • The p element holds paragraph text
  • The br tag represents a line break between two blocks of content

CSS

  • There are 3 ways to style a webpage with CSS: inline, internal, external CSS Style Sheet

  • Inline styles are made within the HTML file, for a particular element
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file
  • External CSS style sheets keep the CSS rules in a separate file from the HTML
  • External CSS style sheets are preferred, as they make design changes easier
  • External CSS style sheets must be linked to the html file via a 'link' element

  • A selector defines the element or attributes to which the rules, or declarations, will apply
  • Declarations contain two components: CSS property and the value of the property
  • Margin property adds a buffer space outside of the element
  • Padding property adds space inside the element, so increasing the padding will make the element bigger
  • CSS Box Model: Every html element is a box
  • Box model => content | padding | border | margin
    • Deep Dive on CSS Box Model

  • Using the wildcard (*) in the CSS file, will apply the CSS declarations to all elements in html file
  • CSS declarations can also be applied to html tags, or specific elements via the 'class' attribute
  • To specify a class attribute in the CSS style sheet, a period (.) must be placed before the class attribute name
  • Do Not Repeat Yourself (DRY), by combining multiple elements with the same style separated by a comma (,)
  • Some ways to assign colors in CSS is via semantic reference (such as blue), or specifying the color's hexadecimal code

Git

  • Git is a version control tool, and GitHub is a cloud-based hosting service for managing repos
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages all changes made in current directory
  • git commit -m "commit message": pushes staged changes to local branch
  • git push origin featureBranch: uploads commited local repo content to remote repo
  • git pull origin master: downloads latest changes from master branch
  • git fetch: fetches all changes made in remote repo, but does not download them

JavaScript

    JavaScript Basics
    JavaScript Data Types

  • A variable is a named container for storing data in our code
  • To declare a variable, use the 'var' keyword + the unique variable name
  • Utlize the 'Console . log ()' method to print information or code ouput into the console
  • To link the .js file to the html file, utilize the 'script' html tag in the body section of the html file
  • Control flow is the order in which a computer executes code in a script
  • JavaScript executes code in consecutive order, from the first statement at the top of the page, down to the last statement at the bottom of the page

  • JavaScript Conditional statements are a way to control the order in which code is executed, or if it is even executed at all
  • Using conditional statements such as if statements allows us to interrupt control flow
  • Falsy values: 0, -0, 0n, " ", null, undefined, Nan, boolean value 'false'
  • Truthy values: All other values, including boolean value 'true'

  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal
  • An array is a single variable that is used to hold a group of related data
  • Arrays can contain multiple data types such as strings, numbers, and boolean values
  • The location of a data item in an array is called an index
  • JavaScript is zero-indexed, meaning arrays start with 0
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • For loop structure: for keyword ( 3 conditional statements separated by semi colons) and the code block in curly brackets to execute
