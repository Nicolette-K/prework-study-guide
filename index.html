HTML

  • There are two main elements in an html file, head and body
  • The head and body parent elements hold many child elements

  • The head element contains information about the webpage.
  • The head element is a container for meta elements which provide the browser the info needed to render the web page
  • The head element content is not visible to the user

  • The body element represents the visible content shown to the user.
  • The ul, ol, il elements are used for unordered lists, ordered lists, and list items respectively
  • Headers are represented using the h1-h6 tags, h1 being the largest text size and h6 being the smalllest
  • The id attribute is important in identifying elements
  • The p element holds paragraph text
  • The br tag represents a line break between two blocks of content

Git

  • Git is a version control tool, and GitHub is a cloud-based hosting service for managing repos
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages all changes made in current directory
  • git commit -m "commit message": pushes staged changes to local branch
  • git push origin featureBranch: uploads commited local repo content to remote repo
  • git pull origin master: downloads latest changes from master branch
  • git fetch: fetches all changes made in remote repo, but does not download them
